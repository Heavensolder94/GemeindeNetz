# GemeindeNetz

GemeindeNetz ist ein moderner Frontend-Prototyp für eine Plattform, die christliche Gemeinden miteinander vernetzt.

## Aktueller Stand: v3

- hochwertige responsive Landingpage
- Bereiche **„Wir bieten“** und **„Wir suchen“**
- Registrierung und Anmeldung
- persönliches Nutzer-Dashboard
- Gemeinde- und Profildaten
- Beiträge erstellen und löschen
- lokale Datenspeicherung im Browser über `localStorage`
- Apple-inspiriertes Glassmorphism-Design mit Animationen und Bildern

## Lokal starten

Die Datei `index.html` kann direkt im Browser geöffnet werden. Es ist aktuell kein Build-Prozess notwendig.

## Technischer Hinweis

Diese Version ist ein Frontend-Prototyp. Konten und Beiträge werden nur im jeweiligen Browser gespeichert. Als nächster Schritt ist ein echtes Backend mit Supabase vorgesehen, inklusive Authentifizierung, PostgreSQL-Datenbank, Storage und Zugriffsregeln.
